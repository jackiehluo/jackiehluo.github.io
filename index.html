About Me

Hi, there! My name is Jackie Luo, and I'm a junior at Columbia University studying computer science and philosophy. I'm on the board of the Columbia Organization of Rising Entrepreneurs (CORE) and part of ADI Labs, a student-led initiative to build open-source projects for the Columbia community.

In CORE, I launched Ignite@CU for undergraduate designers, developers, and entrepreneurs in New York City, with talks from companies like Shapeways, Fifty-Three, Bonobos, and more. I also led design and marketing for #StartupColumbia and the #StartupColumbia Challenge, the university's first annual entrepreneurship festival and venture competition with a $50,000 prize pool for students.

This summer, I was an engineering intern at Percolate, building the mobile automation framework for our iOS and Android apps. I was also selected as one of twenty women to attend Square Code Camp, a week-long immersion program with coding workshops, leadership sessions, and a hackathon. If you're in New York, get in touch!

Projects


No New Friends

I developed an app to connect strangers at Columbia over coffee. Inspired by Tea with Strangers, No New Friends is meant to give students who want to meet people outside of their normal circles the opportunity. The app is built in Flask and implements user registration and login with a PostgreSQL database and the Facebook Graph API for importing profile pictures. Users must have @columbia.edu email addresses to register and can then view each others' profiles and social media links and contact each other. No New Friends is currently deployed on Heroku, and you can view it on GitHub.


Sentiment

I worked on an ADI Labs team of four to build Sentiment, an app that analyzes the sentiment at Columbia based on students' online comments. As a back-end developer, I created a web scraper to get all comments on Bwog, a popular campus publication, for the last day and run it through sentiment analysis. The SQL database stores information for each day, like the top comments for that week and the sentiment at that point in time. We're currently working on mapping changes in mood onto major events on campus. You can view the project on GitHub.


GitHub Stats

I'm in the progress of creating GitHub Stats, a project to find the most active users on GitHub by number of contributions and length of streak. This year, I decided to start committing to GitHub daily because I wanted to learn more and learn quickly, and I became curious about the users with the most involvement on the site. The project calls the GitHub API for all usernames and then implements a web scraper and analysis tool to pull the user's contributions and streak lengths (information that's not available through the API), filtering the greatest. I'm currently working on optimizing speed, automating more of the processes, and eventually visualizing the data I get from millions of developers. The project is, of course, on GitHub!
